# AI Sales Team — Referência Técnica

## Repositório de origem

https://github.com/zubair-trabzada/ai-sales-team-claude

## Uso estratégico no Nexa OS

Este repositório deve ser usado como referência para estruturar uma camada de agentes comerciais especializados para a Nexa Malls.

## Aplicações práticas

- SDR Agent
- Qualification Agent
- Follow-up Agent
- Proposal Agent
- Meeting Prep Agent
- Executive Commercial Agent

## O que aproveitar

- lógica de múltiplos agentes comerciais;
- divisão de responsabilidades por etapa do funil;
- qualificação de leads;
- geração de cadências comerciais;
- preparação de reuniões;
- suporte à criação de propostas;
- análise de pipeline.

## Adaptação para a Nexa Malls

A lógica comercial deve ser adaptada para prospecção B2B imobiliária, com foco em:

- redes em expansão;
- heads de expansão;
- franquias;
- operadores de varejo, serviços e alimentação;
- strip malls;
- lojas satélites;
- âncoras;
- BTS e locações comerciais.

## Regra de implementação

Não copiar o código integral sem análise técnica e jurídica. Usar como benchmark de arquitetura, prompts, fluxo de agentes e lógica comercial.
