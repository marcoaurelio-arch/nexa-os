# NEXA OS

Sistema Operacional Inteligente da Nexa Malls.

## Arquivos

| Arquivo | Uso |
|---|---|
| `CONFIG.json` | Configuracao estruturada do agente, metas, ativos, equipe e integracoes |
| `PROMPTS.md` | Prompts mestres e prompts por modulo |
| `PLAYBOOK_IMPLANTACAO.md` | Plano de implantacao por sprints |
| `ROTINA_DIARIA.md` | Rotina diaria de briefing, checkpoint comercial e fechamento |
| `BACKLOG_GITHUB.md` | Issues e labels recomendadas para GitHub |
| `SPEC_DASHBOARD_VERCEL.md` | Especificacao do dashboard executivo MVP |
| `CHECKLIST_DRIVE_CALENDAR.md` | Checklist para Google Drive e Google Calendar |

## Ordem de uso recomendada

1. Validar `PROMPTS.md`.
2. Revisar `CONFIG.json`.
3. Implantar `CHECKLIST_DRIVE_CALENDAR.md`.
4. Executar `ROTINA_DIARIA.md`.
5. Abrir issues de `BACKLOG_GITHUB.md`.
6. Seguir `PLAYBOOK_IMPLANTACAO.md`.
7. Desenvolver `SPEC_DASHBOARD_VERCEL.md`.

## Regra central

O NEXA OS pode preparar, organizar, analisar e recomendar. Mensagens externas, decisoes comerciais sensiveis, alteracoes financeiras relevantes e compromissos juridicos exigem aprovacao humana.
